# GUI_tickets

### Sales and Receipt Tracking System

This is a project developed in **C#** using **Entity Framework Core** and **MySQL**. 

##  Features
- Receipt and user management.
- Modular architecture.

## Technologies Used
- .NET 10 (Windows)
- Entity Framework Core
- MySQL / Pomelo MySqlConnector
- Git for version control

## Configuration
To run this project, you need to create an `appsettings.json` file in the root directory with the following format:
\`\`\`json
{
  “ConnectionStrings”: {
    “DefaultConnection”: “Server=localhost;Database=your_database;Uid=your_username;Pwd=your_password”
  }
}
\`\`\`
